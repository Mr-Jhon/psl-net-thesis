# PSL-Net Thesis

Reimplementation of **Crowd Counting and Individual Localization Using Pseudo Square Label (PSL-Net)**.

## Objective

- Reimplement PSL-Net
- Train on public crowd counting datasets
- Deploy model to Android
- Use as thesis project

## Framework

- Python
- PyTorch
- OpenCV

- Android Studio
